1. Introduction aux erreurs et exceptions

En programmation, une exception est un événement qui interrompt le flux normal d'exécution d'un programme.

Exemple simple :

print(10 / 0)  # ZeroDivisionError

2. Types d'erreurs en Python

Erreur Description Exemple
ZeroDivisionError Division par zéro 10 / 0
ValueError Valeur inappropriée int("abc")
FileNotFoundError Fichier introuvable open("fichier_inexistant.txt")

3. Bloc try-except

try:
    age = int(input("Entrez votre âge : "))
except ValueError:
    print("Veuillez entrer un nombre valide !")

4. Blocs else et finally

try:
    fichier = open("donnees.txt")
except FileNotFoundError:
    print("Fichier non trouvé")
else:
    print("Contenu :", fichier.read())
    fichier.close()
finally:
    print("Opération terminée")

5. Lever des exceptions

def calcul_racine(x):
    if x < 0:
        raise ValueError("Le nombre ne peut pas être négatif")
    return x ** 0.5

try:
    calcul_racine(-4)
except ValueError as e:
    print("Erreur :", e)

6. Gestionnaires de contexte

# Avec gestionnaire de contexte
with open("fichier.txt") as f:
    contenu = f.read()
    # Le fichier se ferme automatiquement

# Equivalent sans gestionnaire de contexte
try:
    f = open("fichier.txt")
    contenu = f.read()
finally:
    f.close()

7. Exemple pratique complet

def calcul_moyenne(notes):
    try:
        if not notes:
            raise ValueError("La liste ne peut pas être vide")
        
        return sum(notes) / len(notes)
    
    except TypeError:
        print("Erreur : La liste doit contenir des nombres")
    except ZeroDivisionError:
        print("Erreur : La liste ne peut pas être vide")
    except Exception as e:
        print(f"Erreur inattendue : {e}")
    finally:
        print("Fin du calcul")

# Test
notes = [15, 18, "20", 12]
moyenne = calcul_moyenne(notes)
