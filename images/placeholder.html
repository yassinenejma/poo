Le Concept du Polymorphisme

La capacité d'objets de classes différentes à répondre au même message de façon spécifique à leur type

Polymorphisme Ad Hoc

Surcharge d'opérateurs et de fonctions selon le type ou le nombre de paramètres

Polymorphisme de Sous-type

Redéfinition des méthodes dans les classes dérivées via l'héritage

Polymorphisme Paramétrique

Utilisation de types génériques pour manipuler différents types de données

Bonnes Pratiques

Principes et techniques pour utiliser efficacement le polymorphisme
